# 🎓 CampusIQ — AI Campus Assistant

An AI-powered university chatbot that helps students instantly find information about campus rooms, teacher offices, labs, and facilities using natural language.

## ✨ Features
- Ask questions in plain English
- Find classrooms, labs, and buildings instantly
- Know teacher office locations and availability
- Library timings and location
- Beautiful green & purple chat UI
- Real-time AI responses

## 🛠️ Built With
- Python
- Groq API (LLaMA 3.3)
- HTML, CSS, JavaScript

## 🚀 How to Run
1. Clone the repo
2. Install dependencies: `pip install groq`
3. Add your Groq API key in `chatbot.py`
4. Open `app.html` in your browser
